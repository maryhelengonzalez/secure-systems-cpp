# secure-systems-cpp
Advanced C++ systems, concurrency, and security simulations


## Key Concepts Demonstrated
- Thread-safe worker pool using `std::thread`, `std::mutex`, and `std::condition_variable`
- Job scheduling and graceful shutdown
- Race condition demonstration and fix using atomic operations
- Unit testing of concurrent code
- Continuous Integration with GitHub Actions
